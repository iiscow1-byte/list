# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

You can Report vulnerabilities through the
[discord server](https://discord.gg/thehub) or by DM'ing Jonas758
(Jonas758#1230). Responses to vulnerability reports usually take
around 500 hours to more than a year, depending on how difficult fixing a
vulnerability is.
